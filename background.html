Backround

This is a shorthand property and sets all the background properties in one declaration.

Possible Values:

  • attactchment
  • color
  • image
  • repeat
  • position

Background-attatchment

The background-attatchment property specifies whether a background image is fixed with regard to the viewport or scrolls along the containing block.

Possible Values:

  • Scroll (default) - The background iage will scroll with th erest of the content.
  • Fixed - The backgroud image will remain statoinary as the rest of the content is scrolled.

background-image: url("img_tree.png");

backgroud-repeat: no-repeat;

backgroud-attatchment: fixed;

Backgroud-color

Sets the background color of an element.

Possible Values:

  • transparent (default)
  • color (colorname, HEX Value, etc.)

Backgroud-color: pink;

Background-image

background-image property sets the backgroud image of an element.

Possible Values:

  • none (default)
  • url

body { background-image: url("iamge.png");}

backgroud-position

Specifies the initial position of a baackground image.

Possible Values:

  • top
  • right
  • bottom
  • left
  • combination
  • center
  • percentage
  • lenghts

body{ background-position: top; background-position: bottom;}

Backgroud-repeat

Specifies hwo a background iamge will repeat itself within an element.

Possible Values:

  • repeat default - tilted, repeating the image both horizontally and vertically
  • repeat-x repeats the image horizonally only.
  • repeat-y repeats the image vertically
  • no-repeat - does not repeat the image

body{ background-image: url("gradient_bg");

background-repeat: repeat-x, }
